Model Formula Reference

ENV 1. Environment

Atmospheric conditions from weather inputs

OutputFormula
air_densityrho = 1.225 * (P/1013.25) * (288.15/T)
headwindV_w * cos(wind_dir - heading)
temp_at_altT_gnd - 0.0065 * alt
Feeds: Airframe (rho), all models (temp)

AERO 2. Airframe

Aerodynamic forces & flight mode selection

OutputFormula
drag_Nq*S*(Cd0 + CL^2/(pi*AR*e))
wing_lift_Nq*S*CL*wing_frac
rotor_lift_reqW - wing_lift
power_parasiticD * V
Feeds: ICE Engine, Rotor

PROP 3. ICE Engine

Fuel-burning cruise engine with altitude derating

OutputFormula
P_availableP_max * alt_factor * temp_factor
fuel_flowBSFC*(1+0.15*(1-thr)) * P
Alt derate: 3%/1000ft (NA), 1.5%/1000ft (EFI)
Feeds: Fuel System, Generator

PROP 4. Fuel System

Fuel state, endurance, dynamic weight

OutputFormula
endurance_hrusable_fuel_g / fuel_flow_g_hr
range_nmiendurance * cruise_speed_kts
mass_currentm_empty + payload + fuel_rem
Feeds: Vehicle mass for downstream

PROP 5. Rotor (VTOL)

Electric lift rotor thrust & power

OutputFormula
thrust_NCt * rho * n^2 * D^4 * N_rot
power_WCp * rho * n^3 * D^5 * N_rot
rpmsqrt(T / (Ct*rho*D^4)) * 60
Ct_eff = Ct0*(1-0.3*mu^2) Cp_eff = Cp0*(1+0.5*mu^2)
Feeds: Motor Electrical

PROP 6. Motor Electrical

BLDC motor current, voltage, efficiency

OutputFormula
current_Atorque / Kt * N_rotors
voltage_VRPM/Kv + I*R_winding
P_elec_WV * I * N_rotors
Feeds: ESC Losses

PROP 7. ESC Losses

Speed controller conduction + switching losses

OutputFormula
esc_loss_WI^2*R_esc + P_motor*sw_%
total_elec_WP_mot + ESC + compute + avi
Feeds: Battery

PROP 8. Battery (1RC Thevenin)

Voltage sag, SoC, electrical endurance

OutputFormula
V_terminalOCV*Ns - I*(R0+R1+Rw)
endurance_min(E_Wh / P_W) * 60
OCV(s) = 3.0 + 2.035s - 5.325s^2 + 12.74s^3 - 12.88s^4 + 4.63s^5 R_temp = R_nom * (1 + 0.005*(25-T))
Feeds: Generator

PROP 9. Generator

ICE starter-generator & hybrid boost

OutputFormula
gen_avail_WP_cont * (1 - engine_load)
charging_Wmin(surplus, charge_rate)
hybrid_boostactive if load > 85%
Feeds: Avionics, Compute

AVI 10. Avionics

Navigation filter & geotag accuracy

OutputFormula
pos_unc_msqrt(GPS^2 + EKF^2)
geotag_errsqrt(pos^2 + (V*dt)^2 + att^2)
GPS noise: L1=2.5m L1/L2=1.5m RTK=0.02m PPK=0.01m
Feeds: Compute

AVI 11. Compute

AI inference under thermal constraints

OutputFormula
eff_latencybase_latency / throttle_factor
eff_fpsmax_fps * throttle_factor
T_junc = T_amb + P * 3.0 C/W throttle = 1 - (T_junc - T_limit)/30
Feeds: Comms, Identification

AVI 12. Comms

RF link budget & bandwidth-limited quality

OutputFormula
link_marginTx + 2*G - FSPL - Rx_sens
comp_QF90 * (BW_avail / BW_req)
FSPL = 20*log10(d_km) + 20*log10(f_MHz) + 32.45
Feeds: Image Quality

PERC 13. GSD

Ground sample distance from camera geometry

OutputFormula
gsd_cm_px(sw * alt) / (fl * px_w) * 100
pixels_on_tgttarget_size / gsd_m
fov_h_deg2*atan(sw / (2*fl))
Feeds: Motion Blur, Image Quality, ID

PERC 14. Motion Blur

Image smear from platform motion

OutputFormula
smear_pxsqrt((V*t_exp/GSD)^2 + vib^2)
safe_speedblur_budget * GSD / t_exp
Feeds: Rolling Shutter, Identification

PERC 15. Rolling Shutter

RS distortion from sequential readout

OutputFormula
rs_skew_pxV * t_readout / GSD
rs_wobble_pxomega * t_readout * fl_px
rs_totalsqrt(skew^2 + wobble^2)
Feeds: Image Quality, Identification

PERC 16. Image Quality (GIQE 4.0)

NIIRS-equivalent composite image utility

OutputFormula
system_mtflens * 0.64 * sinc(blur) * comp
snr_db20*log10(S/sqrt(S+N^2))
NIIRS = 10.251 - 3.32*log10(GSD_in) + 1.559*log10(RER) - 0.344/SNR GSD in INCHES (cm/2.54). RER = 0.5 + 0.5*MTF
Feeds: Identification

FINAL 17. Identification

P(identification success) - FINAL OUTPUT

P(id) = accuracy_nominal * pixel_density_factor * (1 - blur_penalty) * (1 - compression_penalty) * (1 - latency_penalty) * image_utility_score
FactorFormula
blur_penmin(blur * 0.15, 0.5)
comp_penmin((90-Q)/10 * 0.05, 0.3)
px_densitymin(1, POT / (res*0.1))
ood_riskdegradation / ood_threshold

Envelope Solver

Evaluates all 17 models at every (speed, altitude) grid point to produce mission feasibility surfaces

SurfaceLogic
Feasibilityaero_ok AND engine_ok AND fuel_ok AND blur_ok AND battery_ok AND ceiling_ok
ID ConfidenceModel 17 output at each grid point
Endurancefuel_endurance_hr from Model 4 at each grid point
Mission P(success)fraction of grid where feasible AND identification_confidence >= threshold
Grid: 20x20 (speed x altitude). Power demand: P = D*V / eta_prop. ISA density: rho = 1.225*(T/288.15)^4.2561
